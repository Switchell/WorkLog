## 🎯 О проекте

AXIS ERP PRO — это комплексная система для учёта рабочего времени команды. Разработана для фрилансеров и небольших команд, которым нужно:

- 📊 Отслеживать время работы сотрудников
- 💰 Считать выручку, затраты и прибыль
- 📈 Визуализировать данные в графиках
- 📱 Получать отчёты в Telegram
- 🔐 Давать клиентам доступ к статистике

## 🚀 Быстрый старт

### 1. Установка

```bash
# Клонируйте проект
cd work-tracker

# Настройте .env
nano .env
```

### 2. Запуск

```bash
docker-compose -f docker/docker-compose.yml up -d
```

### 3. Готово!

- **Приложение**: http://localhost/
- **Grafana**: http://localhost:3001
- **Логин**: `admin`
- **Пароль**: `2213`

## 🌐 Развёртывание

### Локально
```bash
./scripts/start.sh
```

### На сервере
```bash
# 1. Скопировать проект
scp -r work-tracker user@server:/home/user/

# 2. Настроить .env
nano .env

# 3. Запустить
chmod +x scripts/*.sh
./scripts/start.sh
```
