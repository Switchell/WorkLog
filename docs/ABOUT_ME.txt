## Наш проект: WorkLog

**WorkLog** — система учёта рабочего времени (Python/Streamlit/Docker/PostgreSQL/Grafana)

### Структура проекта:
```
work-tracker/
├── src/           # app.py, app_client.py, backup.py
├── docker/        # docker-compose, Dockerfile, nginx.conf
├── scripts/       # setup.sh, start.sh, stop.sh
├── docs/         # README.md, DEPLOY.md, kwork_projects.txt
└── backups/      # SQL бэкапы
```

### Особенности:
- Nginx reverse proxy на порту 80
- Grafana на порту 3001
- Клиентский портал на /client
- SSL/HTTPS настроен (через certbot)

### GitHub:
https://github.com/Switchell/WorkLog

## Ключевые решения

- **Пагинация убрана** — слишком тяжёлая
- **Кэширование** — @st.cache_data(ttl=300)
- **UNIQUE constraint** на (Date, Sotrudnik)
- **Nginx** вместо прямых портов для админ-приложения
